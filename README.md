# open-browser [![PyPI version](https://badge.fury.io/py/open-browser.svg)](https://badge.fury.io/py/open-browser) ![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)

> 使用 Chrome 浏览文件

平时文本文件默认的打开方式都设置为 vscode，所以命令行下打开 html 文件比较麻烦，所以写了个小工具用于在命令行使用 Chrome 打开文件。

### 安装

``` shell
$ pip install open-browser
```

### 使用

``` shell
$ open --help
usage: open [-h] [-v] URL

Open url with your browser.

positional arguments:
  URL            the url you want to open with browser

optional arguments:
  -h, --help     show this help message and exit
  -v, --version  displays the current version of `open`
```
